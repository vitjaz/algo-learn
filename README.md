# 🧠 Algo Learn

**Algo Learn** — интерактивная платформа для изучения алгоритмов и структур данных через наглядные визуализации.

🌐 [Read in english](./README.en.md)

---

## 💡 О проекте

Понимание алгоритмов — ключевой навык для любого разработчика. Но учить их по сухим учебникам скучно и неэффективно. **Algo Learn** превращает изучение алгоритмов в интерактивный процесс: вы видите, как алгоритм работает шаг за шагом, прямо в браузере.

### Что внутри

- 🔍 **Бинарный поиск** — пошаговая визуализация деления массива пополам
- 🫧 **Сортировка пузырьком** — анимация попарного сравнения и обмена элементов
- 🔀 **Сортировка слиянием** — визуализация divide-and-conquer подхода
- ⚡ **Быстрая сортировка** — анимация выбора опорного элемента и разбиения
- 🌙 Тёмная и светлая темы
- 🌍 Двуязычный интерфейс (русский / английский)
- 📱 Адаптивный дизайн для мобильных устройств

### Реализованные алгоритмы

| Алгоритм             | Категория  | Лучший   | Средний  | Худший   | Память  |
| -------------------- | ---------- | -------- | -------- | -------- | ------- |
| Бинарный поиск       | Поиск      | O(1)     | O(logn)  | O(logn)  | O(1)    |
| Сортировка пузырьком | Сортировка | O(n)     | O(n²)    | O(n²)    | O(1)    |
| Сортировка слиянием  | Сортировка | O(nlogn) | O(nlogn) | O(nlogn) | O(n)    |
| Быстрая сортировка   | Сортировка | O(nlogn) | O(nlogn) | O(n²)    | O(logn) |

## 🚀 Локальный запуск

### Вариант 1: Node.js (разработка)

#### Предварительные требования

- **Node.js** ≥ 18
- **pnpm** (рекомендуется) или npm/yarn

#### Установка

```bash
# Клонируйте репозиторий
git clone https://github.com/vitjaz/algo-learn.git
cd algo-learn

# Установите зависимости
pnpm install

# Запустите dev-сервер
pnpm dev
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

### Вариант 2: Docker (продакшен)

#### Предварительные требования

- **Docker** с поддержкой Docker Compose

#### Запуск

```bash
# Клонируйте репозиторий
git clone https://github.com/vitjaz/algo-learn.git
cd algo-learn

# Соберите и запустите контейнер
docker compose up -d --build
```

Откройте [http://localhost:3000](http://localhost:3000) в браузере.

#### Полезные команды

```bash
docker compose logs -f       # Логи
docker compose down           # Остановить
docker compose up -d --build  # Пересобрать и запустить
```
